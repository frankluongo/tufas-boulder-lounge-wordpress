# WordPress Theme & Plugin Starter Kit
A Theme/Plugin Starter Kit, made in 2026. Inside should be all the files for developing, building and deploying a WP Theme and WP Plugin out of one repo.

## Getting Started

1. Change `.env.sample` to `.env` and update values with the correct names
2. Create a `[plugin-name].php` file inside the `src/plugin` directory.
3. Create whatever other files you need for your theme / plugin.
4. Start development with the `npm run watch` command
